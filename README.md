#### [Meridian — Bloomberg-style trading terminal](https://github.com/Hugues2601/Meridian)
Full-stack terminal with a C++ Black-Scholes pricing engine, live options chain with 3D implied vol surface, AI investment advisor (Random Forest + FinBERT + Claude synthesis), and a FRED macro dashboard covering CPI/PCE, Sahm Rule, and yield curve across 8 maturities.

`C++` `Python` `React` `TypeScript` `FastAPI`

---

#### [Forward start options under stochastic volatility](https://github.com/Hugues2601/ForwardStartOptions)
Master thesis — calibrated stochastic volatility models with PyTorch + CUDA on live API-extracted options data. Greeks computed via Automatic Adjoint Differentiation (AAD). Full PnL explained/unexplained analysis to assess calibration accuracy.

`PyTorch` `CUDA` `Python` `AAD`

---

#### [Market making with Heston models](https://github.com/Hugues2601/HestonMarketMaking)
Interactive options pricing and market-making tool. Fourier-based pricing for Heston, Bates, and Double Heston via characteristic functions. Bid-ask spread engine + Newton-Raphson implied volatility solver.

`Python` `Streamlit` `NumPy` `SciPy`
